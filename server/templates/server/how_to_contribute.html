How to Contribute

Checklist Template Licensing

When contributing a Checklist Template to the Orange Button Checklist Registry, you are agreeing to publish it under the Apache 2.0 License.

You can contribute your checklist template by emailing it to info@oballiance.org in one of the following formats:

Orange Button Checklist Template JSON.

Note that the Checklist Registry will automatically generate UUIDs for the ChecklistTemplateID and ChecklistTemplateMaintainerID. Click the "Download OB JSON" button on the example Orange Button Checklist Template page to download an example JSON file.

CSV of Orange Button ChecklistTemplates with Questions.

Sending a CSV of Checklist Templates and Questions is trickier since the CSV format does not support nested lists and objects directly. Please see the example Checklist Template CSV files to see how Checklist Templates can be serialized using the CSV format. We are happy to provide advice for how you should construct your Checklist Template CSV files; however, we strongly recommend using the Orange Button JSON format to avoid ambiguity.

Thank you for contributing your Checklist Template to the Checklist Registry!
